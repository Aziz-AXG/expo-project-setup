# Expo Project Setup with:

- TypeScript
- expo
- expo-router
- i18next
- @tanstack/react-query
- zustand
- react-native-mmkv
- react-native-reanimated

## Run the Setup

```
git clone https://github.com/Aziz-AXG/expo-project-setup
```

```
cd expo-project-setup

```

```
pnpm expo prebuild
```

update the sdk.dir in android/local.properties

```
pnpm i
```

```
pnpm run start
```
